# 🚚 Smart Logistic Automation Test
โปรเจกต์ทดสอบระบบอัตโนมัติ (Automation Testing) สำหรับระบบ Smart Logistic ERP

## 🛠 เครื่องมือที่ใช้ (Tech Stack)
* **Framework:** Playwright
* **Language:** JavaScript
* **Tools:** VS Code, Node.js

## 🧪 ขอบเขตการทดสอบ (Test Scenarios)
สคริปต์นี้ครอบคลุมการทดสอบฟังก์ชันสำคัญ เช่น:
1. ตรวจสอบหน้าแรกและชื่อแบรนด์ (Landing Page)
2. ตรวจสอบการสลับ Tab ไปหน้าเช็คพัสดุ (Navigation)
3. ทดสอบการเข้าสู่ระบบ (Login) ทั้งแบบรหัสถูกและรหัสผิด

## 🚀 วิธีการรันเทส (How to run)
1. ติดตั้ง Library: `npm install`
2. รันสคริปต์: `npx playwright test`
3. ดูรายงานผลการเทส: `npx playwright show-report`
